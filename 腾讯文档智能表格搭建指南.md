# 扭捏niunie · 腾讯文档智能表格搭建指南

> 跟着这篇指南，10-15分钟搭好你店铺的完整管理系统。
> 搭完后发个链接给店员，她微信里就能直接开单。

---

## 第一步：创建智能表格

1. 电脑浏览器打开 [腾讯文档](https://docs.qq.com)
2. 登录你的微信/QQ账号
3. 左侧边栏点 **「+」** → 选 **「智能表格」**（不是普通表格）
4. 创建后会看到一个空白的多维表格
5. 左上角标题处改成 **「扭捏niunie 店铺管理」**

---

## 第二步：创建「产品库」数据表

### 2.1 新建数据表

1. 左侧边栏底部点 **「+ 新建数据表」**
2. 命名为 **「产品库」**
3. 把默认的第一列改名为 **「货号」**（点击列名直接改）

### 2.2 添加所有字段

点击每列的 **「+」** 号或在列头右侧点箭头 → 插入列，依次创建以下字段：

| 列名 | 字段类型 | 说明 |
|------|----------|------|
| 货号 | **文本** | 默认已有，改成这个名字 |
| 款式编码 | **文本** | 同一款式多个SKU共享 |
| 商品名称 | **文本** | 产品全名 |
| 颜色 | **文本** | |
| 尺码 | **文本** | |
| 售价 | **数字** | 选数字类型，小数位0 |
| 库存 | **数字** | 整数 |
| 产品图片 | **附件** | ⭐关键！选「附件」类型 |
| 淘宝链接 | **URL** | 超链接类型 |

**如何修改列类型：** 点击列名右侧的 **「▼」箭头** → 选「更改字段类型」→ 选择对应类型

### 2.3 导入产品数据

**方法一：用CSV导入（推荐，快速）**

1. 我已经帮你生成了 `店铺智能管理系统_飞书导入.csv` 文件
2. 把它从工作区下载到电脑桌面
3. 在智能表格中点左上角 **「⋮」** → **「导入」** → 选 **CSV**
4. 上传CSV文件，映射对应列名
5. 导入后检查数据是否完整（应该有277行）

**方法二：从Excel粘贴（备选）**

1. 打开我生成的 `店铺智能管理系统.xlsx`
2. 切到「📦产品库」表
3. 选中A3到J280的所有数据，Ctrl+C复制
4. 回到腾讯文档智能表格的产品库，点第一行空行，Ctrl+V粘贴

### 2.4 上传产品图片 ⭐

> 图片已经帮你全部下载好了，在 `产品图片` 文件夹里（234张）

**电脑端操作：**

1. 打开「产品库」数据表
2. 找到「产品图片」这一列（附件类型）
3. 点击第一个单元格 → 出现 **「+」号**
4. 点 **「上传文件」** → 从 `产品图片` 文件夹选对应图片
5. 重复操作277次（建议：一次性打开文件夹，按货号对应上传）

**批量上传技巧：**
- 腾讯文档支持 **拖拽上传** —— 直接把图片文件拖到附件单元格里
- 可以按住Shift多选图片，一次性上传到多个单元格

**手机端操作（如果店员需要帮忙上传）：**
- 打开链接 → 点附件列 → 选相机图标拍照/选图 → 上传

## 第三步：创建「每日销售」数据表

### 3.1 新建数据表

1. 左侧边栏底部点 **「+ 新建数据表」**
2. 命名为 **「每日销售」**

### 3.2 添加字段

| 列名 | 字段类型 | 说明 |
|------|----------|------|
| 销售日期 | **日期** | 当天日期 |
| 成交人 | **文本** | 填店员名字 |
| 货号 | **关联** | ⭐关联到「产品库」表的「货号」字段 |
| 商品名称 | **引用** | ⭐引用自「产品库」的「商品名称」 |
| 颜色 | **引用** | 引用自「产品库」的「颜色」 |
| 尺码 | **引用** | 引用自「产品库」的「尺码」 |
| 标价 | **引用** | 引用自「产品库」的「售价」 |
| 实收金额 | **数字** | 手动填写实际收款 |
| 数量 | **数字** | 默认1，可改 |
| 小计 | **公式** | 见下方公式 |
| 大单奖励 | **公式** | 见下方公式 |
| 备注 | **文本** | |

### 3.3 设置关联字段 ⭐核心

**创建「货号」关联字段：**
1. 添加列 → 类型选 **「关联」**
2. 弹窗中选择关联到 **「产品库」** 表
3. 关联字段选 **「货号」**
4. 确定后，这列就可以搜索选择了

**创建引用字段（自动带出产品信息）：**
1. 添加列 → 类型选 **「引用」**
2. 引用源选刚才创建的「货号」关联列
3. 选择要引用的字段（商品名称/颜色/尺码/售价）
4. 每个字段重复以上步骤创建一列

> **效果：** 店员在「货号」列输入或搜索货号后，商品名称、颜色、尺码、标价全部自动带出！

### 3.4 设置公式

**小计公式：**
1. 添加列 → 类型选 **「公式」**
2. 输入公式：
```
IF(OR([实收金额]="",[数量]=""),0,[实收金额]*[数量])
```
> 意思是：如果实收或数量为空就显示0，否则=实收×数量

**大单奖励公式：**
1. 添加列 → 类型选 **「公式」**
2. 输入公式：
```
IF([实收金额]>=2000,20,IF([实收金额]>=1000,10,0))
```
> 单笔≥2000奖20，≥1000奖10，否则0

### 3.5 创建「日结统计」视图

1. 点击表名右侧的 **「+ 新建视图」**
2. 选 **「看板视图」**，按「销售日期」分组
3. 或选 **「表单视图」**（这个推荐！⭐）

**表单视图 = 手机开单界面：**
- 店员打开链接看到的就像一个表单页面
- 填日期 → 搜货号选产品 → 填实收金额 → 提交
- 提交后自动保存到表格中
- **这才是你想要的开单体验！**

**设置表单视图：**
1. 点 **「+ 新建视图」** → 选 **「表单」**
2. 可以拖拽调整字段顺序，建议：
   - 销售日期（第一）
   - 成交人（第二）
   - 货号（第三）
   - 实收金额（第四）
   - 数量（第五）
   - 小计、大单奖励、备注放后面
3. 可以给「实收金额」设置必填

## 第六步：创建「薪资计算」数据表

| 列名 | 字段类型 | 说明 |
|------|----------|------|
| 日期 | 日期 | |
| 星期 | 公式 | `WEEKDAY([日期])` |
| 当班人员 | 文本 | 填名字，多人用逗号隔开 |
| 当班人数 | 公式 | `LEN([当班人员])-LEN(SUBSTITUTE([当班人员],",",""))+1` |
| 店铺日销售额 | 数字 | 手动填当日总销售 |
| 提成池 | 公式 | 见下方 |
| 人均提成 | 公式 | `IF([当班人数]=0,0,[提成池]/[当班人数])` |
| 店员A大单奖励 | 数字 | 手动填 |
| 店员B大单奖励 | 数字 | 手动填 |
| 店员A日薪档 | 数字 | 填260/280/300 |
| 店员B日薪档 | 数字 | 填260/280/300 |
| 备注 | 文本 | |

**提成池公式（阶梯提成，全额制）：**
```
IF([店铺日销售额]<=3000,[店铺日销售额]*0.02,
IF([店铺日销售额]<=6000,[店铺日销售额]*0.025,
IF([店铺日销售额]<=10000,[店铺日销售额]*0.03,
[店铺日销售额]*0.04)))
```

> 意思：日销≤3000按2%，≤6000按2.5%，≤10000按3%，超过按4%

## 常见问题

**Q：附件/图片字段怎么上传图片？**
A：电脑端拖拽图片文件到单元格即可。手机端点击附件列的「+」号可以拍照或从相册选。

**Q：公式不生效怎么办？**
A：腾讯文档智能表格的公式语法跟Excel略有不同，字段名用方括号 `[字段名]`。如果报错，检查字段名是否完全匹配。

**Q：关联字段搜不到产品？**
A：确保产品库里有数据，且关联的字段名正确。搜索时输入货号的部分字符也能模糊匹配。

**Q：手机端体验怎么样？**
A：表单视图最适合手机操作。建议店员收藏链接，微信里一秒打开就能填单。

**Q：提成阶梯要改怎么办？**
A：直接修改薪资表里的提成池公式，改比例数字就行。
