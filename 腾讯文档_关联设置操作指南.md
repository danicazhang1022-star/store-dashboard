# 腾讯文档智能表格 · 关联设置操作指南

> 这份指南教你如何把各工作表之间的「货号」设为**关联字段**，设好之后在任意表输入货号关键字（比如"2511"或"小猫"），就会自动弹出产品供你选择。

---

## 📌 当前问题

现在各表里的「货号」是普通文本，互相之间没有关联。所以你输入"2511"或"小猫"时不会弹出产品列表，只能手动打字。

**解决方法：把「货号」改成「关联」类型，关联到产品库。设好后：**
- 输入 "2511" → 自动弹出 NN2511 开头的所有产品
- 输入 "小猫" → 自动弹出"爱读书的小猫"等包含"小猫"的产品
- 选中后，商品名称、颜色、尺码、售价会**自动填入**

---

## 第一步：设置「每日销售」表的关联

这是店员每天用得最多的表，先设这个。

### 1.1 删除现有的文本「货号SKU」列

1. 打开智能表格 → 点左下角进入 **「每日销售」** 表
2. 找到 **「货号SKU」** 这一列
3. 点击列名右侧的 **「▼」箭头**
4. 选 **「删除列」**

> 💡 删掉的是空列（还没销售记录），不会丢失数据。如果已经有销售记录了，先看最后面的"注意事项"。

### 1.2 创建「关联」类型的货号列

1. 点击任意列右侧的 **「+」** 号 → 添加新列
2. 列名填：**「货号」**
3. 字段类型选：**「关联」**
4. 弹窗中设置：
   - **关联表**：选 **「产品库」**（或叫"智能表1"）
   - **关联字段**：选 **「货号SKU」**
5. 点确定

### 1.3 验证关联是否成功

1. 在新创建的「货号」列中随便点一个空单元格
2. 应该会出现一个搜索框
3. 输入 **"2511"** → 看看是否弹出 NN2511 开头的产品
4. 输入 **"小猫"** → 看看是否弹出"爱读书的小猫"

> ✅ 如果弹出了产品列表，说明关联设置成功！

### 1.4 添加「引用」字段（自动带出商品信息）

关联字段只能选货号，但如果还想让商品名称、颜色、尺码、售价自动显示，需要添加**引用字段**：

1. 点列右侧 **「+」** → 新增列
2. 列名：**「商品名称」**
3. 类型选：**「引用」**
4. 弹窗中：
   - **引用源**：选刚才创建的 **「货号」** 关联列
   - **引用字段**：选 **「商品名称」**
5. 点确定

重复以上步骤，再创建3个引用列：

| 列名 | 类型 | 引用源 | 引用字段 |
|------|------|--------|----------|
| 颜色 | 引用 | 货号（关联列） | 颜色 |
| 尺码 | 引用 | 货号（关联列） | 尺码 |
| 标价 | 引用 | 货号（关联列） | 售价 |

> ⚠️ 注意：如果原来已经有「商品名称」「颜色」「尺码」等文本列，先删掉旧的再创建引用列。

**完成后效果：** 在「货号」列选了一个产品后，「商品名称」「颜色」「尺码」「标价」自动填入！店员只需选货号 + 填实收金额即可。

## 📸 操作图示说明

### 关联字段在哪里选？

添加新列时，类型列表里找 **「关联」**（图标是一条连接线的样子）：

```
文本
数字
日期
单选
多选
├── 关联    ← 选这个
引用
公式
...
```

### 引用字段在哪里选？

添加新列时，类型列表里找 **「引用」**：

```
...
关联
├── 引用    ← 选这个
公式
附件
...
```

引用字段的设置弹窗会让你选：
- **引用源** = 选你刚才创建的关联列（如「货号」）
- **引用字段** = 选要自动带出的产品信息（如「商品名称」）

## ⚠️ 注意事项

### 如果表中已经有数据怎么办？

如果「每日销售」表里已经有手填的销售记录（货号是文本），不能直接删列。操作步骤：
1. 先添加新的「关联」类型的「货号」列（不要删旧的）
2. 新列会显示为空，新录入的销售记录用新列
3. 旧记录的货号只能手动重新选择一次（点单元格 → 搜索 → 选中）
4. 全部迁移完后再删除旧的文本列

### 产品库的表名可能不同

如果你的产品库表不叫「产品库」而是叫「智能表1」或其他名字，关联时选对应的表就行。可以在左侧边栏看到所有表的名称。

### 关联字段支持模糊搜索

- 输入 "NN" → 弹出所有NN开头的产品
- 输入 "2511" → 弹出货号含2511的产品
- 输入 "猫" → 弹出名称含"猫"的产品
- 支持中文和数字混合搜索
